Machanized Ink

Exploring a new paradigm for graphic vector drawing UI

Machanized Ink is what I call an interface for users to create, edit, modify vector graphics with a pointing-device centric aproach.

Unlike typicical drawing interfaces, which have evolved from desktop GUI systems, this system narrows user inputs to pointing device on screen (mouse left mouse button click-drag-release or stylus press-drag-release). Avoided is the requirement for the "hunting & pecking" type inputs where users are to construct and manipulate vector graphics by pressing an assortment of buttons and other GUI controls. The net effect is a UI which is more natural in feel, intuitive, and in-line with traditional physical art mediums. A digitized art medium, as such, combines the properties of ink and drawing tools for a machine of graphic drawing.
